# Сравнение моделей бинарной классификации для предсказания покупок (Логистическая регрессия vs KNN)

## Описание задачи
Даны данные пользователей социальной сети: возраст, пол и приблизительная заработная плата. Необходимо предсказать, купит ли пользователь рекламируемый товар. 
Задача решается методами бинарной классификации.

## Используемые технологии
- Python
- Pandas, NumPy 
- Scikit-learn 
- Matplotlib

## Методы и результаты
В ходе работы было обучено и оценено **4 варианта моделей** для поиска наилучшего решения:

| Модель | Масштабирование (StandardScaler) | Точность (Accuracy) |
| :--- | :--- | :--- |
| Логистическая регрессия | ❌ Без масштабирования | **83.3%** |
| Логистическая регрессия | ✅ С масштабированием | 76.6% |
| K-Nearest Neighbors (KNN) | ❌ Без масштабирования | 78.3% (k=13) |
| K-Nearest Neighbors (KNN) | ✅ С масштабированием | 76.6% (k=4) |

**Итог:** Лучший результат показала **логистическая регрессия без масштабирования признаков**, достигнув точности **83.3%**. 
В ходе работы также был найден оптимальный гиперпараметр `k` (количество соседей) для метода KNN методом перебора значений от 1 до 14.
